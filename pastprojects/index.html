Past projects

Stupid Questions - An Alexa Skill (2018)

A fun project, try this out with Alexa yourself: "Alexa, Ask me a stupid question!". A simple Node.js application that returns a random question from an array to Alexa to speak. FUN FACT: 93 people in USA said this to Alexa in a month without any advertisement whatsoever!

Github Users' Performance Calculator (2018)

A RESTful application based on github official API and developer tools. It can calculate and compare the performance of different users on the basis of entered repositories. It also creates the very easy to read report that you can also get delivered at your email. Email service disabled for security reasons; Code is still there to see!

Dynamic Sustainable Wayfinding (2018)

An android application that could provide directions from one point to another using multiple modes of transport calculating and comparing distance, cost, timings. It was specifically made for Dublin and connected directly to Dublin Bikes', Bus's, Luas's and Dart's APIs.

GiftingJewels.com: e-Commerce Platform (2016-2017)

Website and Android app for selling jewelry. Made from scratch using no 3rd party services. Implemented self-written security algorithms and connected with a payment gateway. It showed success as a real world business application bringing sales. [Closed: 16 December 2017]

Multi-client Chat Server (2017)

A chat server that could connect multiple users in multiple groups. Different versions in C, C++, Java, using both UDP and TCP.

IoT Based Automatic Greenhouse (2017)

The Nordic semiconductor nrf52 development kit was coded to emulate the sensor data of a farm environment like temperature, wind speed, etc and send it to ThingsBoard, on which a user can monitor, automate, and control the sensors accordingly.

Pac-man (2017)

An Artifical Intelligence foundational project developed by Berkeley, University of California. The challenge was to make an AI player that could win the game in even the hardest levels.

JPWaterSupply.in (2017)

A simple static website for a water tankers business in India.

Practitioner's Choice (2016)

An Android application that could diagnose(guess) a disease by comparing the symptoms selected by the user, and could suggest remedies and tips regarding the same.

PostKuchBhi.com (2015) - Translated in English means "PostAnything"

A WordPress classifieds website that did not require any sort of registration or verification. All the posts are also automatically posted on website's Facebook page and Twitter handle. Had to shut down because of spam!

Kool Skool (2014)

Probably the dumbest name ever! A school time personal project - java application that used school's computer lab's central database to read and write posts, messages, bios, etc of the users.
